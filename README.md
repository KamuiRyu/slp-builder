# SLP Builder

Builder frontend para montar, calcular e compartilhar builds de shinobi.

## Tecnologias

- React
- TypeScript
- Vite
- TailwindCSS
- React Hook Form
- Zod

## Como rodar

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
```

## Compartilhamento

As builds são serializadas em Base64 diretamente na URL. O projeto não usa backend, banco de dados ou autenticação.
